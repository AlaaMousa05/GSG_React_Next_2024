# GSG React & Next.js 2024-2025

A collection of React and Next.js learning projects and exercises built during the GSG training program in 2024-2025.

## Projects Included

- **1_Students App** - A student management application
- **Add-form-hook** - Custom React hooks for form handling
- **Calculator** - A calculator built with React
- **Task Tracker** - A task tracking application
- **TodoAppReact** - A todo list application built with React

## Technologies

- **React** - JavaScript library for building user interfaces
- **Next.js** - React framework for production
- **TypeScript** - Type-safe JavaScript

## Getting Started

Each project folder contains its own implementation. Navigate to any project folder to explore and run the application.
